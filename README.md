# contacts-app


#### Client 
- Angular 4

##### Run Client

* Localhost
```
ng serve
http://localhost:4200/
```

* Web API
```
ng serve --environment=dev
```

* Local Storage
```
ng serve --environment=local
```

#### Server 
- C# .NET Core Web API (Visual Studio 2017)
```
http://contactsapp20170606010329.azurewebsites.net/api
```

#### Cordova 
- Android

* Run on Android device
```
cordova run android
```
